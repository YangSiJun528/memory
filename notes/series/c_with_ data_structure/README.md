## 배경
개발에 대한 기초적인 부분을 공부하기 위해 C언어로 여러 자료구조를 구현해보고 싶다는 생각이 들었다.

- 왜 C언어인가? 
	- 고수준 언어중에서 저수준에 가깝다.
		- 추상화 수준이 적어서 컴퓨터 내부 동작 원리를 생각하면서 개발하기에 더 좋다.
	- 지금까지도 사용율이 높고, 알아서 나쁘지 않다.
		- 요즘 Go나 Rust 같은 언어에 관심을 가지고 있고, C Like 언어는 워낙 많기 때문에, C언어 문법이나 기능을 조금이나마 배워보고 싶다.
